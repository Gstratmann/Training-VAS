Feedback & data
Values here exclude today's entries — they appear only from your next session onward, so you can't reverse-engineer a slider position. Skipped lines are simply absent, never counted as zero.

Each line is a nominal 100 mm; the value is the mark's position as 0–100, valid on any screen. Scoring convention: 100 = most performance-favorable, 0 = least, on every scale.
